Modal Test: Scrolling

Github Issue: #2602

This page tests that when a modal component is opened over top of a scrollable space, scrolling does not occur underneath of the modal. This behavior would be most prevalent in iOS environments.

Scroll this page to the middle, where the modal's trigger button is present, and tap the button to open. After the modal is opened, try to scroll, and close the modal. After closing the modal, the contents of the main body area should not have scrolled.
